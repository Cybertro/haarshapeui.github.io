Introduction

Thank you for purchasing our mobile application template. If you have any questions that are beyond the scope of this help file, please feel free to email us via our user page contact form here.

Food Delivery - React Native Template

Food Delivery is component based application template for React Native. Source code has been written in JavaScript and the final application can be built both for Android and iOS platforms. The application has more than 25 components, 50 screen container layouts and the visual design can easily be customized.

Version: 2.0.0.

Last update: April 19, 2020.

Release date: September 27, 2019.

Author Team: AppDrops

Email: dev.appdrops@gmail.com

Getting Started

This section will help you walk through the minimum required setup to run Food Delivery project.

Required Environment

Important

In order to build the application, please make sure you have already install React Native and run the simulator successful base on React Native CLI Quickstart guide, it will help you install and build your first React Native demo app. The instructions are a bit different depending on your development operating system, and whether you want to start developing for iOS or Android.

Important

In order to build the application, please make sure you have already install Expo base on Expo CLI Quickstart guide, it will help you install and build your first Expo demo app.

Installing and Running the Project

After your system has met the previous requirements, you can build and run the application as follows:

  • Unzip the folder you have downloaded from the marketplace. Food Delivery - React Native Template project is in folder "FoodDelivery_Pro"

  • In terminal, navigate to the folder "FoodDelivery_Pro" where you unziped the files

  • Run the following command:

                        
                      npm install
                      # or with yarn
                      # yarn install

  • Link native dependencies on iOS, make sure you have Cocoapods installed. Then run:

                          
                        cd ios
                        pod install
                        cd ..

  • Open simulator or connect your mobile device via USB

  • Start react packager / server

                        
                      react-native start
                      # you can also use npm or yarn
                      # npm start

  • Run instructions for Android. Open new terminal tab and run command:

    react-native run-android

  • Run instructions for iOS

                          
                        react-native run-ios
                        # or with Xcode
                        # open FoodDelivery_Pro/ios/FoodDelivery_Pro.xcodeproj in Xcode and hit the Run button

  • If everything is set up correctly, you should see your new app running in your simulator (or mobile device) shortly.

  • Unzip the folder you have downloaded from the marketplace. Food Delivery - React Native Template project is in folder "FoodDelivery_Expo"
  • In terminal, navigate to the folder "FoodDelivery_Expo" where you unziped the files

  • Run the following command:

                          
                        npm install
                        # or with yarn
                        # yarn install

  • Start a development server

                          
                        expo start
                        # or with npm
                        # npm start

  • Install the Expo client app on your iOS or Android phone and connect to the same wireless network as your computer. On Android, use the Expo app to scan the QR code from your terminal to open your project. On iOS, follow on-screen instructions to get a link.

File Structure

Below is how the project file structure looks like.

App.js
file is the entry point of the application, it renders main navigator (MainNavigatorA is imported by default). 

src
folder contains all the files that make the application, grouped as follows:

src/assets
folder contains all dummy images used in application.

src/components
folder contains subfolders with source code of individual React Native components used throughout the application. Individual components have been described in the separate section of this documentation. 

src/config
folder contains a file in which you can choose one of predefined application themes (color scheme).

src/navigation
folder contains files MainNavigatorA.js and MainNavigatorB.js, import one of these two files into App.js to see the difference in design.

src/screens
folder contains subfolders of predefined screen layouts (containers). Each screen layout has two or more different designs (e.g. AboutUsA.js, AboutUsB.js, ...)

src/theme
folder contains two files: colors.js with predefined color themes in it, and layout.js in which screen layout constants (width, height, margin, padding, ...) are defined.

src/utils
folder contains helper functions used by the application.

Customization

Short instructions for adjustments and changes.

1) Choose application design type

There are two predefined navigators in our application. MainNavigatorA contains all screen layouts characterized by a light background with flat buttons (as described above in Screens section of this document). While MainNavigatorB contains all screen layouts with a gradient backgrounds, colorful appbar and contained buttons. Choose the one that suits you best. MainNavigatorA is imported by default.

App.js

2) Choose application color theme

Go to ./src/config/index.js. There are three predefined color themes: "jade", "redOrange" and "blueberry". Color theme "jade" is selected by default.
